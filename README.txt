# Archive information #
# =================== #

This data archive is structured in a way that it is clear how the various files are used to generate 
specific elements from the manuscript. This means that most data, analysis scripts, and output files 
are ordered into folders that correspond to specific figures from the main text or supplementary 
material. These folders are:

/Figure_1
 Contains SVG files, PDF and PNG versions of Figure 1. The .SVG files can be opened using e.g. 
 Inkscape, which is freely available online through https://inkscape.org.

/Figure_2
 Contains the code ("analysis.R") used to generate the data used in Figure 2, as well as additional 
 folders containing the data and .pdf and .png versions of the figure. A separate README.txt is
 included with further details. 

/Figure_3
 Contains the code ("analysis.R") used to generate the data used in Figure 3, as well as additional
 folders containing the data and .pdf and .png versions of the figure. A separate README.txt is 
 included with further details. 

/Figure_4
 Contains the code ("analysis.R") used to generate the data used in Figure 4, as well as additional 
 folders containing the data and .pdf and .png versions of the figure. A separate README.txt is 
 included with further details. 

/Figure_5 
 Contains the code ("analysis.R") used to generate the data used in Figure 5, as well as additional 
 folders containing the data and .pdf and .png versions of the figure. A separate README.txt is 
 included with further details. 

/Figure_6 
 Contains SVG files, PDF and PNG versions of Figure 6. The .SVG files can be opened using e.g. 
 Inkscape, which is freely available online through https://inkscape.org.

/manuscript
 Contains the final manuscript, and a .doc of the final submitted version of the manuscript.

/supplementary_figures 
 Contains a folder for each supplementary figure (Figures S1-S6), each of which also includes 
 a README.txt file with further details.
