./analysis.R 
This file runs the analysis required to generate Supplementary Figure S5. It generates two datasets 
and saves these as data_S5A.txt and data_S5B.txt, contained in the folder ./data. The analysis file 
also generates the combined figure and saves these as .pdf and .png files, named 2024_12_30_S5.pdf 
and 2024_12_30_S5.pdf, contained in the folder ./figure_files.
