./analysis_script.R 
This file runs the analysis required to generate Supplementary Figure S1. It runs through 9 
different combinations of the parameters r and z. 

The analysis script writes the data to the data file data_S1.txt and saves the figure as a .pdf and 
.png, named 2024_12_30_Figure_S1.pdf and 2024_12_30_Figure_S1.png.
